Doxygen DITA Edition User Guide Readme
======================================

To build the user guide you need to download the DITA Open Toolkit http://dita-ot.sourceforge.net.

From ant_scripts directory run the DITA OT build file build_doxygen_dita_edition_user_guide.xml for your chosen format,

e.g.:

C:\#path_to_Doxygen_DITA_edition_user_guide_source#\ant_scripts>ant -f build_doxygen_dita_edition_user_guide.xml dita2xhtml

A CHM build of the User Guide is pre-built and distributed with Doxygen DITA Edition 0.1
